### How to Use

#### Opening the File

1. Download the `.xlsm` file from this repository
2. Open Microsoft Excel
3. Navigate to File > Open and select the downloaded file
4. If prompted about macros, click "Enable Content" or "Enable Macros" to activate the VBA functionality

#### Running Macros

**Option 1: Using the Developer Tab**
1. Go to the Developer tab in Excel
2. Click "Macros" button
3. Select the desired macro from the list
4. Click "Run"

**Option 2: Using Keyboard Shortcuts**
- Press `Alt + F8` to open the Macro dialog box
- Select the macro you want to run
- Click "Run"

**Option 3: Using Buttons or Form Controls**
- If the workbook contains buttons or controls, simply click on them to execute the associated macros

#### Viewing/Editing VBA Code

1. Press `Alt + F11` to open the VBA Editor (Visual Basic Editor)
2. In the Project Explorer pane (left side), expand the project tree
3. Double-click on modules, forms, or worksheet objects to view their code
4. Make changes as needed (if you have editing permissions)
5. Press `Alt + F11` again to return to Excel

### Security Considerations

**Important:** Only open and enable macros in files from trusted sources. Malicious VBA code can potentially harm your computer or compromise your data.

- This file is from an academic project and should be safe to use in an educational context
- Always scan files for viruses before opening
- Consider opening the file in a protected or sandboxed environment if you're unsure

### Troubleshooting

**Macros Don't Run:**
- Check that macros are enabled in Excel's Trust Center settings
- Verify that the file is in .xlsm format (not .xlsx)
- Ensure you clicked "Enable Content" when opening the file

**VBA Editor Won't Open:**
- The Developer tab may not be visible. Enable it through File > Options > Customize Ribbon
- Check if VBA is installed with your version of Excel

**Runtime Errors:**
- Some macros may require specific data or worksheet structures
- Verify that all required worksheets and named ranges exist
- Check if external references or add-ins are needed
